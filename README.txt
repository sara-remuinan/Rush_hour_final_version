Rush Hour Shift - simple terminal version

Files:
- main.py: starts the game
- game/constants.py: enums and fixed values
- game/pieces.py: cards and vehicles
- game/actions.py: simple action objects
- game/state.py: current game state
- game/setup.py: starting position and deck
- game/display.py: terminal rendering
- game/rules.py: legal actions and state updates
- game/input_parser.py: asks the human to choose an action
- game/engine.py: game loop

How to run:
python main.py

Notes:
- This is a simple hardcoded terminal implementation.
- It does not include any AI player.
- It is a playable approximation of Rush Hour Shift rather than a full official reproduction.
